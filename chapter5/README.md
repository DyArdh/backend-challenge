# Challenge Chapter 4

#### Installation Steps:

1.  Rename .env.example to .env and setting credentials data.
2.  NPM Install

```
npm install
```

3.  Create Database

```
npm run createdb
```

4.  Run Migrations

```
npm run migrate
```

5.  Run Seeders

```
npm run seeder
```

6.  Run API

```
npm run dev
```
